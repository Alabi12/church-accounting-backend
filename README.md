# Church Accounting System - Backend API

A comprehensive, AI-powered church accounting system built with Flask.

## Features

- ✅ Income Management (Tithes, Offerings, Special Collections)
- ✅ Expense Tracking
- ✅ Budget Management
- ✅ Payroll Processing
- ✅ Financial Reporting
- ✅ AI-Powered Analytics
- ✅ Anomaly Detection
- ✅ Giving Pattern Prediction
- ✅ Budget Optimization
- ✅ Multi-currency Support
- ✅ Audit Logging
- ✅ Role-Based Access Control

## Tech Stack

- **Framework**: Flask 2.3
- **Database**: PostgreSQL with SQLAlchemy
- **AI/ML**: scikit-learn, Prophet, TensorFlow
- **Task Queue**: Celery with Redis
- **Authentication**: JWT with 2FA support
- **API Documentation**: Swagger/OpenAPI
